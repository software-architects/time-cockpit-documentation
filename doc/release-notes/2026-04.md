# 📅 April 2026

## ✨ Improvements

- **Configurable Filename for Report Downloads in Lists**  
  The filename used when downloading reports from lists can now be configured. This makes it easier to identify downloaded files without renaming them manually.

- **Display Used Profile in List**  
  Lists now show which formatting profile is currently active. This gives users better insight into how their data is being displayed.

- **"Markdownize" Formatting Profile Info**  
  Formatting profile information now supports Markdown rendering. Descriptions and notes in formatting profiles are displayed with proper formatting.

- **Updated Dialog When Deleting a Time Cockpit User**  
  The default confirmation dialog shown when attempting to delete a user has been revised. It now provides clearer information about the consequences of the action.

- **Right-Click on Group Logo in Lists**  
  Right-clicking on a group logo in a list now opens the context menu, consistent with right-clicking the group row itself.

> **Note:**
> These release notes describe what’s included in the April 2026 release. Availability can depend on rollout rings and tenant configuration.
